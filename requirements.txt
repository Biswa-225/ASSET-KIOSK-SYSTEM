A) Hardware / OS prerequisites (Pi 5 + RC522)
Wiring (SPI0)

RC522 → Raspberry Pi 5

VCC → 3.3V (NOT 5V)

GND → GND

SDA/SS → GPIO8 (CE0)

SCK → GPIO11 (SCLK)

MOSI → GPIO10

MISO → GPIO9

RST → GPIO25 (you can change)

Enable SPI
sudo raspi-config
# Interface Options -> SPI -> Enable
sudo reboot

Install deps
sudo apt-get update
sudo apt-get install -y python3-spidev python3-rpi.gpio
pip3 install mfrc522
